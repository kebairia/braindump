#+title: aspell and emacs
#+filetags: :emacs:aspell:spell_checker:spell:

to use flyspell in [[id:4c72a4b7-ee23-473d-a50d-39060c7cbb9e][emacs]] you must install the following packages on your system
- aspell
- aspell-en
On Arch Linux for example
#+begin_src shell
sudo pacman -S aspell aspell-en
#+end_src
