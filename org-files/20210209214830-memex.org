#+title: memex
#+roam_tags: knowledge records hypertext "knowledge base" "As We May Think"
the memex is a hypothetical electromechanical device described by Vannevar Bush in his essay "As We May Think" 1945.
the memex is like a parallel memory that helps the user to store knowledge.
the user can store all of his books, records ...etc, so that the memex can help him later to extract this records(knowledge) easily.
